# Welcome to the OpenChamp Manual
This technical manual covers the high-level and technical details for the OpenChamp project, with a current focus on the **OpenChampPS API**. It is intended for developers who wish to join the OpenChamp project.

The technical manual has three different kinds of documentation.
- Guides covering installation and configuration.
- Overviews covering a set of Technical Specifications intended to assist understanding it from an outside context.
- Technical Specifications covering in-depth details about a feature or end point

This documentation includes detailed endpoint specifications, request/response schemas, error codes, and practical examples.
